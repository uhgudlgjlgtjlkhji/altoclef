# AltoClef - Minecraft 26.1.2

This mod has been updated to Minecraft 26.1.2 with Fabric.

## Build Status

Check the **Actions** tab for build results. The JAR will be uploaded as an artifact.

## Quick Start

1. Go to [Actions](https://github.com/drmcbride12/altoclef/actions)
2. Click on the latest workflow run
3. Download the `altoclef-26.1.2-jar` artifact
4. Place the JAR in your `.minecraft/mods` folder
5. Launch Minecraft 26.1.2 with Fabric Loader

## Requirements

- Minecraft 26.1.2
- Fabric Loader 0.18.4
- Fabric API 0.149.0+26.1.2
