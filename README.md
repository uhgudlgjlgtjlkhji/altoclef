# Alto Clef

A Minecraft automation mod for Fabric, powered by Baritone pathfinding.

## Features

- Automated pathfinding using Baritone
- Task automation
- Resource management
- Combat assistance

## Installation

### Requirements

- Minecraft 26.1.2
- Fabric Loader 0.18.4
- Fabric API 0.149.0+26.1.2

### Getting the Mod

You can download the latest release from:
- [Releases](https://github.com/drmcbride12/altoclef/releases) - Download pre-built JAR files
- [Actions](https://github.com/drmcbride12/altoclef/actions) - Build artifacts from commits

### Manual Installation

1. Download the `altoclef-26.1.2-fabric.jar` file from the releases or actions
2. Place the JAR file in your `.minecraft/mods` folder
3. Launch Minecraft with the Fabric profile

## Building from Source

```bash
# Clone the repository
git clone https://github.com/drmcbride12/altoclef.git
cd altoclef

# Build the mod (requires Java 25)
./gradlew build

# The compiled jar will be in build/libs/
```
